«Оно тебе надо» — Аукцион вещей с историей
«Оно тебе надо» — это современный лендинг для уникального аукциона вещей, в которые никто не верил. Проект сочетает в себе сложную типографику, нестандартные сетки и многослойную визуальную структуру.

 Технологический стек HTML5: Использование семантических тегов для обеспечения доступности и SEO.

CSS3:

Grid Layout: Построение сложных асимметричных сеток (шапка, секция «Об аукционе»).

Flexbox: Гибкое управление карточками лотов и элементами навигации.

Позиционирование: Реализация оверлеев и декоративных слоев.

Typography: Тонкая настройка межбуквенных интервалов (letter-spacing) и межсловных расстояний (word-spacing) для реализации дизайнерских акцентов.

📐 Ключевые решения проекта
Архитектура лейаута
Центрирование логотипа: Реализовано с помощью CSS Grid и дробных единиц (fr), что позволяет логотипу оставаться в центре страницы независимо от разного объема контента в левой и правой частях шапки.

Адаптивность карточек: Список лотов реализован через flex-wrap и flex-grow: 1, что обеспечивает органичное заполнение пространства при изменении ширины экрана.

Типографика и стиль
Сложные заголовки: В секции cover использованы вложенные <span> для точечного управления интервалом между буквами в слогане.

Кастомные разделители: В навигационном меню использовано свойство list-style: circle для создания уникальных визуальных отсечек между пунктами.

Интерактивные элементы: Сброс дефолтных стилей кнопок и ссылок с заменой на кастомные состояния (прозрачные фоны, специфические границы).

Работа с медиа
Overlay-система: Создан универсальный класс .overlay для наложения полупрозрачных фильтров на изображения, что позволяет сохранять читаемость текста на любом фоне.

Интеграция контактов: Блок адреса оптимизирован для мобильных устройств с использованием протоколов tel: и mailto:.

📂 Структура проекта
index.html — основная разметка страницы.

styles/global.css — сброс настроек и общие стили.

styles/style.css — основные стили проекта.

fonts/ — локальные файлы шрифтов.

images/ — оптимизированные графические ресурсы.

🚀 Запуск проекта
Просто откройте файл index.html в браузере или воспользуйтесь расширением Live Server в VS Code для мгновенного просмотра изменений.
